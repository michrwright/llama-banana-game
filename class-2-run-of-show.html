Teaching order

Beat 1
Turn on the score display
Do this first — so when the class catches a banana they see the points appear straight away.
CONFIG.scoreVisible = true
Beat 2
Rainbow banana
Introduce it, let them catch one, they see +10. Then vote: rainbow or golden?
bananaConfig.specialActive = true
bananaConfig.specialStyle = 'rainbow' // or 'golden'
Beat 3
Bad banana
Introduce it, let them catch one, they see −20. The penalty makes the game strategic.
bananaConfig.badActive = true
Beat 4
Speed ramp
Game gets progressively faster as time goes on. Turn it on and let them feel the difference.
CONFIG.speedRampEnabled = true
Beat 5
Sounds
Press keys 1–6 to preview each sound. Class votes which sound goes on which banana type.
CONFIG.soundsEnabled = true
